Portfolio site for Joel Kennedy
=================

**Contributors:** Grant Kiely, John Gesimondo

Build requirements
----
* SASS
* Compass
* Node & Gulp js

Installation

------------
**SASS & JS:**

* If you are using windows, install ruby.
* gem install compass


* Watch files with gulp
```sh

cd joelfolio
npm install -g gulp
npm install gulp-concat
npm install gulp-uglify
gulp
```
